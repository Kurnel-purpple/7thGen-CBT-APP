# SEATOS Schools Assets

This directory contains client-specific assets for SEATOS Schools.

## Required Files

- **logo.png** - Main logo displayed in the app header (recommended: 200x60px)
- **favicon.png** - Browser tab icon (recommended: 32x32px or 64x64px)

## Instructions

1. Add your SEATOS logo as `logo.png`
2. Add your SEATOS favicon as `favicon.png`
3. Ensure both files are committed to Git
4. Deploy to see your branding in action

## Temporary Fallback

If these files are missing, the app will fall back to the default icon from `assets/icon.png`.
